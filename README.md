# Симулятор энергосети
Проект для генерации и тестирования данных с использованием различных сценариев.




## Установка зависимостей

Well, it's only:

python == 3.12.3

or

```bash
sudo apt install python3
```



## Запуск тестов

Для запуска тестирования используйте следующие команды в директории репозитория.

**1. General тест (Базовое тестирование):**
```bash
python3 main.py data/generators.csv data/test_general.csv
```


**2. Нагрузочный тест:**

```bash
python3 main.py data/generators.csv data/test_high_consumption.csv
```
